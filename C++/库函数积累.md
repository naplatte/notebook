### iomanip库

用于**格式化输入输出流**

| 名称                 | 用途                       | 备注                       |
| -------------------- | -------------------------- | -------------------------- |
| std::setw(n)         | 设置输出宽度               |                            |
| std::setfill(c)      | 设置空白处填充字符         |                            |
| std::setprecision(n) | 设置有效数字或小数点后位数 |                            |
| std::fixed           | 固定小数点后位数           | 常与setprecision()配合使用 |
