### Python随用随记

#### np.random()下的方法

| 函数                | 方法                                        |
| ------------------- | ------------------------------------------- |
| np.random.rand()    | 生成指定形状的[0,1)之间均匀分布的随机浮点数 |
| np.random.shuffle() | 对数组就地进行随机打乱                      |
| np.random.choice()  | 从给定的一维数组中随机选择元素              |
|                     |                                             |
